# QRIS Dynamic Studio

Small Vite-based web app for generating and reading QR codes (QRIS-focused) to help convert static QR codes into dynamic ones.

## Tech Stack
- Vite (Vanilla JS)
- `qrcode` for QR generation
- `jsqr` for QR scanning/decoding

## Requirements
- Node.js (LTS recommended)

## Development
- Install: `npm install`
- Run dev server: `npm run dev`
- Build: `npm run build`
- Preview build: `npm run preview`
